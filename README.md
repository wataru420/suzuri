## develop

suzuri made by node-webkit.If you wanna hack.Please fork me.

### how to run

* run on Mac:
```
cd suzuri
npm install
npm install nodewebkit -g
npm install bower -g
bower install

open -n -a node-webkit app
```
* run on Linux or Windows:
```
nw suzuri/app
```
